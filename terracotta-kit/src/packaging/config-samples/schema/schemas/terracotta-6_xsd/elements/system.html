The 'license' element requires a 'location' and 'type' to be specified. If you omit the 'license' element, Terracotta will run in a limited 'initial' mode, which limits server runtime and limits the maximum number of simultaneously-connected clients. DEPRECATED: license file must be installed to Terracotta installation directory.
